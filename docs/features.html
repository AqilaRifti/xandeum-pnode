Features

Explore the comprehensive features of Xandeum pNode Analytics for monitoring the decentralized storage network.

Dashboard Overview

The main dashboard provides a real-time overview of the entire pNode network:

  • Network Stats - Total nodes, online/offline count, total storage capacity
  • Storage Metrics - Used vs available storage across the network
  • Version Distribution - Breakdown of node software versions
  • Uptime Statistics - Average uptime and reliability metrics

pNode Monitoring

Monitor individual pNodes with detailed metrics:

Node Status

  • Online/Offline - Real-time status based on last heartbeat
  • Public/Private - Whether the node is publicly accessible
  • Version - Current software version (e.g., 0.8.0, 0.7.3)

Storage Metrics

  • Storage Used - Current storage consumption in bytes
  • Storage Total - Total available capacity
  • Storage Committed - Committed storage allocation
  • Usage Percentage - Visual indicator of storage utilization

Network Information

  • IP Address - Node's network address and port
  • RPC Port - Remote procedure call port (typically 6000)
  • Last Seen - Timestamp of last heartbeat
  • Uptime - Days since node started

Geographic Visualization

Interactive maps powered by Leaflet and React Leaflet:

  • Node Locations - Visualize pNode distribution globally
  • Clustering - Automatic clustering for dense regions
  • Popups - Click nodes to see detailed information
  • Filtering - Filter map by status, version, or storage

Data Tables

Powerful data tables built with Tanstack Table:

  • Sorting - Click column headers to sort ascending/descending
  • Filtering - Filter by any column value
  • Pagination - Navigate through large datasets
  • Column Visibility - Show/hide columns as needed
  • Export - Export data to CSV or JSON
// Example: Using the data table
import { DataTable } from '@/components/ui/data-table';
import { columns } from './columns';

export function NodeTable({ nodes }) {
  return (
    <DataTable
      columns={columns}
      data={nodes}
      searchKey="pubkey"
      filterableColumns={['status', 'version']}
    />
  );
}

Charts & Visualizations

Recharts-powered visualizations for network analytics:

  • Line Charts - Track metrics over time
  • Bar Charts - Compare node statistics
  • Pie Charts - Version and status distribution
  • Area Charts - Storage utilization trends

Search & Filtering

Advanced search capabilities with Nuqs for URL state management:

  • Global Search - Search by pubkey, IP, or any field
  • Status Filter - Online, offline, or all nodes
  • Version Filter - Filter by software version
  • Storage Filter - Filter by storage capacity range
  • URL Persistence - Filters persist in URL for sharing

Command Palette

Quick navigation with kbar (‚åò+K):

  • Quick Search - Find nodes instantly
  • Navigation - Jump to any page
  • Actions - Trigger common actions
  • Keyboard Shortcuts - Efficient navigation

Dark Mode

Built-in dark theme for comfortable viewing:

  • System Preference - Automatically matches OS setting
  • Manual Toggle - Switch themes manually
  • Persistence - Theme preference saved locally

Error Tracking (Optional)

Sentry integration for production monitoring:

  • Error Capture - Automatic error logging
  • Session Replay - Debug user sessions
  • Performance - Track page load times
