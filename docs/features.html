Features

Explore the comprehensive features included in the Next.js Shadcn Dashboard Starter.

Dashboard Layout

The dashboard comes with a pre-built, responsive layout that includes:

  • Collapsible Sidebar - Navigation with icons and nested menu support
  • Header - User menu, notifications, and search
  • Content Area - Flexible main content with proper spacing
  • Mobile Responsive - Optimized for all screen sizes

Analytics Overview

The dashboard overview page features:

  • Stat Cards - Display key metrics with trend indicators
  • Charts - Line, bar, and area charts using Recharts
  • Parallel Routes - Independent loading states for each section
  • Real-time Updates - Ready for live data integration
// Example: Analytics Card Component
import { Card, CardContent, CardHeader, CardTitle } from '@/components/ui/card';

export function AnalyticsCard({ title, value, change }) {
  return (
    <Card>
      <CardHeader>
        <CardTitle>{title}</CardTitle>
      </CardHeader>
      <CardContent>
        <div className="text-2xl font-bold">{value}</div>
        <p className="text-muted-foreground">{change}</p>
      </CardContent>
    </Card>
  );
}

  • Server-side Pagination - Efficient handling of large datasets

Authentication

Secure authentication powered by Clerk:

  • Sign In/Sign Up - Beautiful, customizable auth pages
  • Social Logins - Google, GitHub, and more
  • Passwordless - Email magic links and OTP
  • Enterprise SSO - SAML and OIDC support
  • User Management - Profile and security settings
// Protecting routes with Clerk
import { auth } from '@clerk/nextjs/server';
import { redirect } from 'next/navigation';

export default async function DashboardPage() {
  const { userId } = await auth();
  
  if (!userId) {
    redirect('/sign-in');
  }
  
  return <Dashboard />;
}

Billing & Subscriptions

Monetization features with Clerk Billing:

  • Pricing Table - Display subscription plans
  • Plan Management - Upgrade, downgrade, cancel
  • Feature Gating - Restrict features by plan
  • Usage Tracking - Monitor resource consumption
// Feature gating example
import { Protect } from '@clerk/nextjs';

export function ProFeature() {
  return (
    <Protect
      condition={(has) => has({ plan: 'pro' })}
      fallback={<UpgradePrompt />}
    >
      <ProContent />
    </Protect>
  );
}

Forms

Type-safe forms with validation:

  • React Hook Form - Performant form handling
  • Zod Validation - Schema-based validation
  • Shadcn Form Components - Beautiful form elements
  • Error Handling - Clear validation messages
// Form with Zod validation
import { z } from 'zod';
import { useForm } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';

const schema = z.object({
  name: z.string().min(2, 'Name must be at least 2 characters'),
  email: z.string().email('Invalid email address'),
});

export function ContactForm() {
  const form = useForm({
    resolver: zodResolver(schema),
  });
  
  return (
    <Form {...form}>
      {/* Form fields */}
    </Form>
  );
}
