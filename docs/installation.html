Installation

Follow these steps to set up Xandeum pNode Analytics on your local machine.

Prerequisites

Before you begin, ensure you have the following installed:

  • Node.js 18+ - Download Node.js
  • Bun (recommended) or npm/yarn - Install Bun
  • Git - Download Git

Step 1: Clone the Repository

git clone https://github.com/AqilaRifti/xandeum-pnode-analytics.git
cd xandeum-pnode-analytics

Step 2: Install Dependencies

# Using Bun (recommended)
bun install

# Using npm
npm install

# Using yarn
yarn install

# Using pnpm
pnpm install

Step 3: Start Development Server

bun run dev

Open http://localhost:3000 to view the dashboard.

That's it!

The dashboard works out of the box with the included pNode data. No additional configuration is required to get started.

Optional: Environment Setup

For additional features like error tracking, create a .env.local file:

cp env.example.txt .env.local

Configure the optional environment variables:

# Sentry Error Tracking (Optional)
NEXT_PUBLIC_SENTRY_DSN=

Available Scripts

Command Description
bun run dev Start development server
bun run build Build for production
bun run start Start production server
bun run lint Run ESLint
bun run format Format code with Prettier

Troubleshooting

Port Already in Use

bun run dev -- -p 3001

Dependency Issues

rm -rf node_modules bun.lock
bun install
