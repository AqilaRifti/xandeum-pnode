What is Xandeum?

Xandeum is a decentralized storage network built on blockchain technology. pNodes (persistent nodes) provide distributed storage capacity across the globe.

pNode Metrics

Metric Description
pubkey Unique node identifier
status Online/offline status
version Node software version
storageUsed Current storage usage
storageTotal Total storage capacity
uptime Node uptime in days
ip Node IP address and port
isPublic Public accessibility flag

Quick Start

git clone https://github.com/AqilaRifti/xandeum-pnode.git
cd xandeum-pnode
bun install
bun run dev

Open http://localhost:3000 to view the dashboard.
